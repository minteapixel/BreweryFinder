# Brewery Finder
A web app where the user can search for local breweries in America. Currently a work in progress.

## Live Demo:
https://minteapixel.github.io/BreweryFinder/

## Built with:
- Vanilla Javascript
- HTML/CSS
- Bulma CSS
- OpenbreweryDB API (https://www.openbrewerydb.org/)
